# led-glow-using-push-button
A simple beginner-friendly Arduino project demonstrating how to use a push button with a pull-up resistor configuration to control an LED. This project helps beginners understand digital input, pull-up resistors, and basic Arduino programming.
## DESCRIPTION:
The system uses a push button as an input device and an LED as the output. When the push button is pressed, the Arduino detects the input signal and turns the LED ON. When the button is released, the Arduino switches the LED OFF. This project is ideal for learning the fundamentals of push button interfacing, digital signal processing, and basic embedded system programming
## FEATURES:
- Turn the LED ON using a push button.
- Turn the LED OFF by releasing the push button.
- Real-time response to user input.
- Beginner-friendly Arduino project.
- Demonstrates digital input and output interfacing.
## COMPONENTS USED:
1. Arduino Uno
2. Breadboard
3. Push Button
4. LED
5. 220Ω Resistor
6. 1kΩ Resistor
7. Jumper Wires
8. USB Cable
