# led-glow-using-push-button
A simple beginner-friendly Arduino project demonstrating how to use a push button with a pull-up resistor configuration to control an LED. This project helps beginners understand digital input, pull-up resistors, and basic Arduino programming.
Components Used:
Arduino Uno-1
Breadboard-1
Push Button-1
LED-1
220Ω Resistor-1
10kΩ Resistor-1
Jumper Wires
USB Cable
